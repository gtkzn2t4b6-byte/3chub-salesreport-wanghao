# 3CHUB 月度看板存档

本文件夹存放 3CHUB 尼日利亚销售部每月最终版看板的独立存档。

每个文件为自包含的单 HTML 文件（含内联 CSS/JS/Chart.js），可随时离线打开查看。

## 文件列表

| 文件名 | 报告日期 | 智能机销量 | 月度目标 | 完成率 | 环比 |
|--------|----------|-----------|---------|--------|------|
| 3CHUB_2026年6月销售看板.html | 2026-06-30 | 32,154 台 | 39,305 台 | 81.8% | -4.3% |

## 使用方式

- 双击任意 .html 文件即可在浏览器中打开
- 每个文件包含4个标签页：数据看板 / 问题汇总 / 价格分析 / 历史趋势
- 文件完全离线，不依赖网络连接

## 说明

- 数据来源：当月每日销售数据 Excel + 目标文件 + 价格表 + 库存 + 成本
- 生成工具：scripts/generate_dashboard.py + scripts/process_data.py
- 当月看板会持续更新，月底最终版存档至此文件夹
